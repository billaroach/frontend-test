1. Описание проекта
Проект представляет собой страницу, содержащую форму с тремя полями: Имя, Номер Телефона и Email
2. Содержание проекта
index.html - верстка страницы
styles.css - файл, содержащий css-стили
ajax.js - js-файл, отвечающий за javascript-валидацию и отправку формы с помощью ajax
validation.js - js-файл с продублированной отдельно функцией валидации validate()
action.php - php-файл для проверки работоспособности; обрабатывает полученные из ajax.js данные и возвращает их на страницу
